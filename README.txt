Pre-requesits:

1) Blender 4.0


Extract the archived files and open the folder containing the files.


Install opencv, mediapipe, bpy and numpy:

Run the following command from an admistrator command prompt opened at the current directory:
"C:\Program Files\Blender Foundation\Blender 4.0\4.0\python\bin\python.exe" -m pip install -r requirements.txt


Run Program:

1) Open the 'character_file.blend' file.
2) Go to 'scripting' tab and click the play button.
3) Go back to 'layout' tab. A new icon should have appeared in the tools bar (Pointer inscribed in a circle). Click it.
4) Click on 'Capture'.
